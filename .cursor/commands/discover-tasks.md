---
description: Discover and auto-intake new development tasks
---

Run the task discovery workflow in `TASK_DISCOVERY_WORKFLOW.md`.

Required behavior:

1. Review required project docs and active task tracker context.
2. Review code hotspots and quality signals for maintainability/UX/feature/CI opportunities.
3. Auto-add up to 10 new tasks to `TASKS.md` with status `[p]` (proposed).
4. Avoid duplicates with existing tasks.
5. Provide a short summary of what was added and which tasks should be promoted next.
